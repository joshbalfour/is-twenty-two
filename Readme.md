# is-twenty-two

An npm package to check if a number is equal to 22.

## Installation

```sh
npm --save i is-twenty-two
```

## Usage

```javascript
const isTwentyTwo = require('is-twenty-two');

isTwentyTwo(22); // true

isTwentyTwo(20); // false

isTwentyTwo("22"); // true

isTwentyTwo("twenty-two"); // true

```
